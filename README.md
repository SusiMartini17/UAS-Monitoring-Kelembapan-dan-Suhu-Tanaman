# Monitoring Kelembapan dan Suhu Tanaman via ThingSpeak

## Deskripsi
Proyek ini merupakan implementasi Internet of Things menggunakan ESP32 untuk memonitor suhu udara, kelembapan udara, dan kelembapan tanah. Data dikirim ke ThingSpeak sehingga dapat dipantau secara real-time.

## Komponen
- ESP32
- DHT22
- Potensiometer (Simulasi Soil Moisture)
- LCD I2C 16x2
- Relay Module
- LED Merah
- LED Hijau
- ThingSpeak
- Wokwi

## Software
- Arduino IDE
- Wokwi Simulator
- ThingSpeak

## Fitur
- Monitoring suhu
- Monitoring kelembapan udara
- Monitoring kelembapan tanah
- Tampilan LCD
- Indikator LED
- Relay otomatis
- Pengiriman data ke ThingSpeak
